# ghpages-portfolio

A simple static portfolio page hosted on **GitHub Pages**.

## Objective

The objective of this project is to provide a lightweight, single-file personal portfolio that can be deployed for free directly from the repository using GitHub Pages — no build step or framework required.

## Tech Stack

- HTML5
- GitHub Pages

## Deploy

Enable GitHub Pages in the repository settings (source: `main` branch, root `/`) and the portfolio will be live at `https://viniciussantos45.github.io/ghpages-portfolio`.
